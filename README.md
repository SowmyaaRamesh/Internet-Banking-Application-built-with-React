# Internet Banking Application

## Tech stack used

The project is built with the following tech stack:

- React - frontend
- Node.js - backend
- MySQL - Database

## Features implemented

The features included in the project are listed below:

- Pin change
- Manage deposits
- View Account statement
- View Balance
- Chatbot assistant
- Manage Insurance policies
